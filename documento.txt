1. OBJETIVO

Desarrollar y visualizar el fractal de Mandelbrot en un shader de fragmento (GLSL),
aplicado sobre la superficie de un objeto 3D, para comprender:
- El mapeo de coordenadas a plano complejo.
- El calculo iterativo del conjunto de Mandelbrot.
- El control visual de ramas e interior mediante mascaras.
- La animacion de aparicion (nacimiento) sin ciclo de muerte.


2. DESCRIPCION DE LA SOLUCION

La solucion implementa el fractal de Mandelbrot directamente en el fragment shader.
Cada fragmento calcula un punto complejo c y ejecuta iteraciones de:

z_{n+1} = z_n^2 + c

Si la magnitud de z supera un umbral, el punto escapa; de lo contrario, se considera
interior del conjunto. Con esta informacion se construyen mascaras visuales para:
- Resaltar ramas externas con un patron senoidal.
- Pintar el interior en negro.
- Hacer una aparicion progresiva (birth) basada en el tiempo.

Ademas, se elimino el ciclo de vida de nacimiento/muerte para que el fractal solo
aparezca una vez y permanezca visible.


3. IMPLEMENTACION

3.1 Estructura principal
- Archivo principal: src/shaders/fragment.glsl
- Funciones clave:
	- cmul(vec2 a, vec2 b): multiplicacion de complejos.
	- mandelbrot(vec2 c): calcula escape suave e indicador de interior.
	- mapToComplex(vec2 p, float zoom, vec2 offset): mapea coordenadas al plano complejo.

3.2 Iteracion del Mandelbrot
- Se utiliza un maximo de 120 iteraciones.
- Se considera escape cuando dot(z, z) > 64.0.
- Para suavizado de color se usa smooth iteration:
	smoothIter = iter + 1.0 - log2(log2(sqrt(r2)))

3.3 Mapeo y proyeccion por cara
- Se calcula una UV por cara usando la normal del fragmento (vNormalW) y posicion
	del objeto (vObjectPos), permitiendo aplicar el fractal de manera coherente sobre
	distintas caras del modelo 3D.

3.4 Control de nacimiento (sin muerte)
- Se usa:
	birth = smoothstep(0.0, 12.0, uTime)
- Esta curva hace que el fractal aparezca gradualmente y luego se estabilice en 1.0.
- Se desacoplo del parametro uSpeed para evitar artefactos al modificar velocidad.

3.5 Mascaras de ramas e interior
- branchMask: detecta estructuras tipo ramas usando un patron periodico sobre escape.
- inside: vale 1.0 para puntos interiores del conjunto.
- blackMask = max(branchMask, inside * birth)

Con esto:
- Las ramas se vuelven negras.
- El interior tambien se vuelve negro.
- Al inicio se evita halo innecesario porque el interior respeta birth.

3.6 Color final
- col = mix(uColor, vec3(0.0), blackMask)
- El color base uColor se mezcla hacia negro segun la mascara combinada.


4. CAPTURAS DE PANTALLA

Se recomienda incluir como minimo:
- Captura 1: estado inicial (inicio del nacimiento).
- Captura 2: fase intermedia (ramas formandose).
- Captura 3: estado final estabilizado.
- Captura 4: detalle del interior negro y ramas negras.

Nota: para cada captura, registrar una breve descripcion y el valor de uTime usado.


5. CONCLUSIONES

- Se logro dibujar el fractal de Mandelbrot en tiempo real con GLSL.
- El uso de mascaras permitio controlar visualmente ramas e interior en negro.
- Se elimino correctamente el ciclo de muerte para mantener una sola aparicion.
- Al desacoplar birth de uSpeed, el shader se comporta de forma estable al variar
	la velocidad.
- La practica consolida conceptos de numeros complejos, iteracion fractal y diseno
	procedural en shaders.
